Moped Dashboard Computer 1:

Tracks speed, mileage, fuel level, distance-to-empty, battery, time, ..., as well as controlling headlamp, tail lamp, signal lights, horn, charging (including rectification, regulation from the bike's alternator, with a USB port to charge your phone while riding.)

It also contains a motion-sensing theft alarm/disabler.

I put about 10,000 km on a bike running this dashboard, with no issues.

All fonts, drivers, etc., written from scratch.

Tuner Technology Demonstration

Early tech demonstration of polychromatic optical guitar tuner. Each LED is precisely set to the desired frequency; if a string is in tune, it will appear stationary. Any out of tune strings will appear to be in motion (the more out of tune, the faster the apparent motion.

You can instantly see if any strings are out of tune (and how badly) with a single strum. For electric guitars, no interfacing with the output signal is required (as per traditional tuners), and the guitar can be quickly tuned even with the amplifier disconnected or off! (Great for tuning on-stage.)

(Since filming the stroboscopic effect at 30 fps wouldn't work, this demo is done by tuning all the strings to multiples of 30hz, hence the odd sounds. The effect is more dramatic viewed directly.)
